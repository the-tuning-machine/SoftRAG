# Visualisation des divergences

Ce dossier contient des scripts pour visualiser différentes divergences et métriques associées à l'analyse de données géométriques.

## Installation des dépendances

Avant d'exécuter les scripts de visualisation, installez les dépendances nécessaires :

```bash
pip install -r ../requirements.txt
```

## Exécution des visualisations

Chaque script Python du dossier (`alpha-div_viz.py`, `complete_div_viz.py`, `psy_viz.py`) peut être lancé directement depuis le terminal :

```bash
python alpha-div_viz.py
python complete_div_viz.py
python psy_viz.py
```

Vous pouvez également passer des arguments à certains scripts si besoin (voir le code source ou la documentation interne de chaque script).

## Remarques
- Assurez-vous d'être dans le dossier `visualization` avant d'exécuter les commandes.
- Les figures générées seront sauvegardées ou affichées selon la configuration des scripts.
